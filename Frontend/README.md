# AlinHub Frontend

This is the React.js + Tailwind CSS frontend for AlinHub, a charity-based digital learning portal.

## Features
- Landing page
- Login/Register
- Role-based dashboards (Student, Teacher, Admin)
- Course browsing and details
- Lesson viewer (PDF, PPT, video)
- Quizzes and assessments
- Discussion forum
- Assignment submission
- Profile and progress tracking
- Responsive navigation

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

## Tech Stack
- React.js
- Tailwind CSS
- React Router
- Context API
- Axios
